# LifeOS

A mindful lifestyle management app built with Flutter & Firebase — centralizing tasks, finances, and personal wellness in one place.

🌐 **[View Landing Page](https://amna107.github.io/lifeos-showcase)**

---

## What it does

- ✅ Task scheduling with a multi-category calendar
- 💰 Financial tracker with recurring bills & budget limits
- 📓 Journaling with media support
- 🌬️ Animated breathing & wind-down exercises
- 🏠 Home management tools

## Built with

- **Flutter** — Clean Architecture (Domain / Data / Presentation)
- **Provider** — state management across decoupled modules
- **Firebase** — Firestore, Auth, Cloud Storage
